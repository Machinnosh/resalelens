# エルメスのバーキンは持ってるだけで毎月2万円の利益が出る。嘘じゃない。データで証明する。

あなたが今、手に持っているそのコーチのバッグ。

1年後、半額以下になります。

残酷？いや、これが現実。

## 「高い買い物=損」は完全に間違い

ちょっとこのデータ見てほしい。

エルメス バーキン 30 → 定価¥1,815,000 → 1年後の相場¥2,087,250

はい。定価より27万円高くなってます。

持ってるだけで。何もしてないのに。毎月¥22,687の利益。

一方、コーチのバッグ → 定価¥80,000 → 1年後¥36,000

55%蒸発してます。月¥3,667の損失。

「安いから」で買ったバッグの方が、実質コストでは高い。

この逆転現象、誰も教えてくれない。

## ブランド品には「見えないコスト」がある

車を買うとき、リセールバリュー気にしますよね？

「トヨタは値落ちしない」「外車は3年で半額」みたいなの、みんな知ってる。

でもブランドバッグで同じこと考えてる人、ほぼいない。

なぜか？

データが散らばってるから。

フリマアプリ、オークション、中古ブランドショップ——取引データは存在する。でもバラバラで、誰も統合して分析してなかった。

## だから作った

「リセールレンズ」というアプリを開発中。

ブランド品のリセール価値を0〜100のスコアで可視化する。

バーキン → スコア95（定価超え。持ってるだけで資産）
シャネル クラシックフラップ → スコア88（8%しか落ちない。実質月¥8,700で使える）
ルイ・ヴィトン ネヴァーフル → スコア72（22%落ち。でもまだ優秀）
コーチ → スコア35（55%落ち。月¥3,667溶けてる）

「実質月額いくらで使えるか」

これがわかるだけで、買い物の意思決定が完全に変わる。

## 狂った事実3選

「1. バーキンは株より儲かる」

S&P500の年間リターンが約10%。バーキンの年間リターンは15%。

バーキンはS&P500に勝ってる。

しかも配当金みたいに「使える」。背負って歩ける株。

「2. シャネルは毎年値上げしてる」

シャネルのクラシックフラップ、2019年は約¥580,000だった。

2026年現在、¥1,305,000。

7年で125%値上げ。インフレ？知らんがな。シャネルがインフレ。

つまり「今買う」が正解という構造。中古価格も連動して上がるから、PRR（再販価値率）は下がらない。

「3. コーチとエルメスの月額差は¥19,000」

コーチ 月あたり損失: ¥3,667
エルメス バーキン 月あたり利益: ¥22,687

差額: ¥26,354

定価は22倍違うのに、実質の月額負担はエルメスの方がマイナス（つまりプラス）。

「安い方が得」という常識が完全に崩壊してる。
